# Infrastructure

Many things to question for building applications.  When starting, these are some guidelines for things to consider.

* Identity Infrastructure
  * Internal
  * External
  * AD Controller
    * Identity
    * Object
    * Permission

* Network Infrastructure
  * Connectivity
    * Continual
    * Occasional
    * Never
  * Latency
  * Boundaries
    * Internal firewalls
    * Subnets

* Storage
  * Database
  * File Systems
    * Network connectivity
